# FastAPI Push Notification API

A simple backend API to demonstrate **Web Push notifications** using **FastAPI** and **Next.js**. This project allows clients to subscribe to push notifications and receive messages from the backend in real-time.

Key features:

- ✅ Subscribe to push notifications via a POST request.
- ✅ Send notifications manually to all subscribers.
- ✅ Health check endpoints for monitoring server status.
- ✅ Full logging of subscription and notification events.
- ✅ CORS enabled for cross-origin requests.
